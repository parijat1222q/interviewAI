# InterviewAI – AI-Powered Interview & Career Companion

InterviewAI is a web-based platform designed to help users practice job interviews, improve communication skills, optimize resumes for ATS systems, and track job applications. It provides realistic AI-driven mock interviews, detailed performance analytics, resume tools, and a job tracker — all in one platform.

---

## Features

### AI Interview Simulator
- Select company and job role  
- Real-time AI recruiter simulation  
- Technical, HR, and behavioral interview modes  
- Voice interaction using WebRTC  
- Adaptive follow-up questions based on answers  

### Performance Analytics
- Sentiment and tone detection  
- Communication and confidence scoring  
- Correct vs. incorrect answer analysis  
- Skill-gap insights  
- Visual dashboards (Chart.js / Recharts)  

### ATS-Friendly Resume Builder
- AI-generated resumes  
- Keyword optimization  
- Professional templates  
- Achievement rewriting suggestions  

### ATS Resume Checker
- Upload existing resumes  
- ATS compatibility score  
- Formatting and keyword analysis  
- Improvement recommendations  

### Learning Hub
- Topic-based resources  
- Articles, guides, and video content  
- Daily question and quizzes  

### Job Tracker
- Track job applications  
- Deadlines, interview dates, and statuses  
- Reminders and notes  

### Leaderboard
- Global performance rankings  
- Badges and recognition  
- Weekly and monthly challenges  

---

## Tech Stack

### Frontend
- React.js  
- TailwindCSS / Material UI  
- WebRTC (voice interaction)

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB  

### AI & ML Components
- OpenAI / Gemini APIs for interview generation and resume optimization  
- Hugging Face models for sentiment and tone analysis  
- Custom ML pipeline for scoring, ranking, and skill-gap detection  

### Additional Tools
- Whisper / Web Speech API (speech-to-text)  
- Text-to-Speech (AI recruiter voice)  
- Chart.js / Recharts (analytics)

## Installation
1. Clone the repository:  
   `git clone https://github.com/parijat1222q/interviewAI.git`
2. Navigate to the project directory:
    `cd InterviewAI`
3. Install dependencies for frontend and backend:
    `npm install` (in both `frontend` and `backend` folders)
4. Set up environment variables (API keys, database URIs, etc.)
5. Start the development server:
    `npm start` (in both `frontend` and `backend` folders)
6. Open your browser and go to `http://localhost:3000`
